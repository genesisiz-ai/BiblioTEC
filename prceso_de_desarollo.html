Detrás de Escena: Así creamos nuestra Biblioteca Virtual

Esta sección documenta el proceso creativo, técnico y colaborativo que llevamos a cabo para hacer realidad el proyecto de la Biblioteca Tercero Básico.

1. Objetivo del Proyecto

El objetivo principal de este proyecto fue diseñar y desarrollar una plataforma web interactiva y accesible que sirva como un repositorio digital para estudiantes de Tercero Básico. Buscamos centralizar recursos de lectura, facilitar la búsqueda de información y fomentar el hábito lector a través de una interfaz amigable e intuitiva.

2. Nuestro Moodboard (Inspiración Visual)

Nota: Nuestra paleta visual se basó en tonos [ej. azules y blancos] para transmitir tranquilidad y concentración, utilizando tipografías sin serifa para garantizar una lectura cómoda en pantallas.

3. Tecnologías Utilizadas

Para construir esta plataforma de manera moderna y eficiente, seleccionamos las siguientes herramientas y lenguajes:

  • HTML5 Para estructurar el contenido de la web de manera semántica, asegurando que sea accesible para los navegadores y lectores de pantalla.
  • CSS3 Para el diseño visual y la adaptabilidad. Lo elegimos para poder crear un diseño responsivo que funcione perfectamente tanto en computadoras como en teléfonos móviles.
  • JavaScript Para añadir interactividad, como [ej. el buscador de libros, el modo oscuro o las alertas dinámicas], mejorando la experiencia del usuario sin recargar la página.
  • GitHub Para el control de versiones. Nos permitió trabajar en equipo simultáneamente, manteniendo un historial seguro de todos los cambios en el código.
  • Vercel Para el despliegue (hosting). Lo elegimos por su integración continua y rápida con GitHub, lo que nos permitió publicar la página en internet de forma gratuita y eficiente.

4. Organización y Distribución del Trabajo

El desarrollo fue un esfuerzo conjunto. Utilizamos metodologías de trabajo colaborativo para dividir las tareas según las fortalezas de cada integrante:

Integrante Rol Principal Responsabilidades Específicas
[Nombre Integrante 1] Desarrollador Front-end Estructuración en HTML y maquetación responsiva en CSS.
[Nombre Integrante 2] Diseñador UI/UX Creación del moodboard, selección de colores y diseño de la interfaz.
[Nombre Integrante 3] Programador Lógico Desarrollo de scripts en JavaScript para las funciones interactivas.
[Nombre Integrante 4] Gestor de Contenido / QA Recopilación de textos, subida a GitHub y pruebas de funcionamiento (testing).

5. Uso de Inteligencia Artificial

Durante el ciclo de desarrollo, integramos herramientas de Inteligencia Artificial como asistentes de apoyo para optimizar nuestro flujo de trabajo. Utilizamos IA principalmente para:

  • Depuración de código (Debugging): Identificar y solucionar errores de sintaxis en CSS y JavaScript de manera más rápida.
  • Inspiración y estructura: Generar ideas base para la redacción de contenidos estáticos y estructurar secciones complejas de HTML.
  • Aprendizaje: Comprender conceptos técnicos avanzados consultando dudas sobre el funcionamiento de flexbox, grid o la configuración de Vercel.

6. Conclusión de la Experiencia

El desarrollo de la "Biblioteca Tercero Básico" ha sido una experiencia de aprendizaje invaluable. Más allá de adquirir habilidades técnicas en programación web y control de versiones, este proyecto nos enseñó la importancia de la planificación, la comunicación efectiva y la resolución de problemas en equipo. Ver cómo un diseño conceptual (moodboard) se transforma en un producto digital funcional, alojado en internet y listo para ser utilizado por nuestros compañeros, nos llena de satisfacción y nos motiva a seguir explorando el mundo de la tecnología.
